# Therapist Chatbot

## Description
A compassionate AI chatbot that simulates a soft, understanding therapist. It detects the user's mood from text input and responds in a comforting, personalized way. Built for mental wellness and experimentation with AI personality modeling.

## Features
- Mood detection from user messages
- Soft, compassionate personality
- Text-based conversation via terminal or web interface
- Easy to extend with additional conversational modules

## Tech Stack
- Python 3.x
- Flask (if web-based)
- Transformers / NLP libraries
- Optional: front-end using HTML/CSS/JS
